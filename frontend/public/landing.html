Stack técnico

Arquitectura da plataforma

⚡

FastAPI + Python 3.12

Backend assíncrono, tipagem estrita, validação Pydantic. 25+ routers, 100+ endpoints. Docs OpenAPI automáticas em /api/docs.

🗄

PostgreSQL 16 + pgvector

Row-Level Security por tenant, pgcrypto para PII, pgvector para embeddings RAG semântico. 24 migrations Alembic.

🧠

Cognira AI (14 módulos)

Transcrição STT, visão computacional, text-to-SQL, RAG, scoring, coaching, relatórios narrativos e wizard de estudos.

📱

Next.js 14 + PWA

40+ páginas, i18n PT/EN/ES/FR, WebRTC P2P, IndexedDB offline, Web Push VAPID e service worker.

🔒

Nginx + TLS 1.3

Reverse proxy, HSTS, CSP, rate limiting por zona. MinIO S3-compatible para fotos. ClamAV antivírus em uploads.

🔐

JWT + 2FA + SSO

Tokens 15min + refresh 7d, TOTP MFA, OIDC/SSO (Azure AD, Okta, Google), RBAC 6 roles + permissões por estudo.

🐳

Docker Compose

7 containers: backend, frontend, postgres, minio, clamav, nginx, coturn. Deploy reprodutível, migrations automáticas.

📋

Audit + Webhooks

Audit log imutável com IP e payload. Webhooks HMAC-SHA256 com delivery log. API Keys com rate limiting por chave.
